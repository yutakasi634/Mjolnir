                UnderdampedLangevin #

                ランジュバン方程式に従い、温度・体積・粒子数一定のシミュレーションを行います。

                \[m\frac{d^2 \bold{r}}{dt^2} = \bold{f}(\bold{r}) - m\gamma\bold{v} + \beta(t)\]

                以下の論文で提案された手法です。

                • J. D. Honeycutt and D. Thirumalai, (1992) Biopolymers
                • Z. Guo and D. Thirumalai, (1995) Biopolymers.

                例 #

                [simulator]
integrator.type = "UnderdampedLangevin"
integrator.remove.translation = true
integrator.remove.rotation    = true
integrator.remove.rescale     = true
integrator.gammas = [
    {index = 0, gamma = 1.0},
    {index = 1, gamma = 1.0},
    # ...
]

                入力 #

                delta_tなどの他のパラメータはSimulatorで設定します。

                • type: 文字列型
                  • Integratorの種類を指定します。"BAOABLangevin"です。
                • remove: テーブル型 (optional)
                  • 系全体の並進・回転速度成分を取り除くことができます。trueにすると取り除かれます。
                  • rescaleがtrueになっていた場合、全体の速度ベクトルをリスケールすることで速度を減算した分の運動エネルギーを補填します。
                  • 省略した場合、全てfalseになります。
                • gammas: テーブルの配列型
                  • 粒子の摩擦係数 \(\gamma_i\) を指定します。
